instacart-scraper

instacart-scraper is a command-line tool for exporting your Instacart order history.

What does instacart-scraper do?

instacart-scraper exports your Instacart order history to JSON, CSV, or text. instacart-scraper outputs order dates, totals, retailers, and full item details.

How do I install instacart-scraper?

You install instacart-scraper with Homebrew:

brew tap mike-vincent/instacart-scraper
brew install instacart-scraper

How do I get my session cookie?

You get the session cookie from your browser. Log into instacart.com, open DevTools, go to Application → Cookies, and copy the value of _instacart_session_id.

How do I use instacart-scraper?

You use instacart-scraper by running it with your session cookie:

instacart-scraper --isid "YOUR_SESSION_ID"

What are some instacart-scraper example commands?

# Export all orders as JSON (default)
instacart-scraper --isid "abc123..."

# Export last 6 months to CSV file
instacart-scraper --isid "abc123..." --months 6 -f csv -o groceries.csv

# Export last 30 days as JSON
instacart-scraper --isid "abc123..." --days 30

# Export orders since a specific date
instacart-scraper --isid "abc123..." --since 2025-01-01

# Export as human-readable text
instacart-scraper --isid "abc123..." -f text

# Export quietly (no progress) and pipe to jq
instacart-scraper --isid "abc123..." -q | jq '.[0]'

# Save all orders to file
instacart-scraper --isid "abc123..." -o all-orders.json
